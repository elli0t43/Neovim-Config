# Neovim Config

My Neovim config with installation script

## Installation
Its still work in progress, I haven't added all plugins and configurations yet.
```bash
cd ~
git clone https://github.com/elli0t43/Neovim-Config.git
cd Neovim-Config
chmod +x install.sh
./install.sh
```
